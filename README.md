# vue2 typescript starter
> This project is a practice project for creating admin page.
> This admin pase consists of a login and main page.

Environment
> Framework
* Vue.js

> Language
* TypeScript

> UI Configuration
* postcss modules
* css grid layout
* css flex layout

## Build Setup

``` bash
# install dependencies
npm install

# run the server on localhost:8080
npm start
```
